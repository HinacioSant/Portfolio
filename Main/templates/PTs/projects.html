    Login
    Click here to go to Login Github

    This is a fairly complete login environment users can use their respectives Google or Github accounts to log-in or register a brand new account. Users can also change password. Users who have forgotten their password can recover their account using the email for the respective account, the user will receive an email with the instructions to create a new password.

    In this application I use as base the django login system with my own validation methods, Google reCAPTCHA and a few other methods to increase security. I also use django-social-auth for dealing with oauth. This application also serves as base to dealing with users within the portfolio.

    MSA
    Click here to go to MSA Github

    MSA is a full blown chat application. It offers a fully encrypted experience, users can make temporary accounts and chat with other users. Every chat message and temporary accounts are deleted every X amount of hours.

    MSA counts with a user to user chat system, notifications when users messages each other, friend/unfriend system, a userlist or search and start chat with a specific user, all in real time.

    Besides Django APScheduler (Periodical user/messages deletion). MSA doesn't make use of any other 3rd party application. All of MSA is built using python and javascript, I do recognize that it can be perceived as a archaic method to build a chat application. But this way does allow me to display a more in depth knowledge, thats why the choice of building this application using these tools.

    TDV
    Click here to go to TDV Github

    TDV is a note taking application. Users can create, edit or delete notes or bullets elements. Users can also use Markdown to style their notes.

    TDV is a fairly simple note taking application, like every other application on this portfolio every element is made by me or heavily changed to fit the purpose of the application, in TDV I use a few Bootstrap elements like tabs present on the sidebar, however this element is used quite distinctively from the original Bootstrap element.

    In this application I use Markdown2 so users can use markdown to style their notes.

    TG2A
    Click here to go to TG2A Github

    TG2A is a gallery application. On the main page users will find a amalgamation of images uploaded by other users. On profile users will find every image they have uploaded and every favorited images.

    TG2A functions as a real time gallery users can add images, those images will appear at the main page of TG2A, users can also favorite and download images.

    In this application I use Pillow for processing the images and creating thumbnails (see models for more in-depth information). I also use Waypoints to create a infinite scroll that fitted the needs of TG2A (see Gallery_management/infinite_scroll and TG2A.js for more in-depth information)
